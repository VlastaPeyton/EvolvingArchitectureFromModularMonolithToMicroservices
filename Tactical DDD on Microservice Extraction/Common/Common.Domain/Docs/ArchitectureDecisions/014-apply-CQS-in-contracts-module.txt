Datum:
	23.8.2025

Problem:
	Contracts module ce najverovatnije rasti dosta i da ce biti vise write nego read to db operacija.

Decision:
	Separate Read and Write DB logic with CQS via MediatR na nivou METODE samo.

Consequences:
	Razdvojeni Read and Write DB procesi unutar METODA, ne na nivou ENDPOINT kao sto je slucaj kod CQRS.
	Ima potencijal da postane CQRS sto je vrhunac.
